## Judging Criteria (from organizer whiteboard)

### Agents Track
- **Reason** — Agent understands goals and context
- **Plan** — Breaks goals into executable steps
- **Tool Use** — Interacts with real external systems
- **Complete** — Delivers finished results

### Key Phrase: "Goal → Break → Interact"

## Demo Script (3 minutes)

### Act 1: The Problem (30s)
"We all spend hours on repetitive web tasks — researching, filling forms,
organizing findings. What if you could just say what you want done?"

### Act 2: Live Demo (2min)
1. Open TXLookup UI
2. Click mic: "Research the top 5 AI companies in Austin, find their founders
   on LinkedIn, and create a Miro board with the results organized by funding stage"
3. Show the agent planning steps in real-time
4. Show browser automation executing (Playwright)
5. Show Miro board being populated live
6. Show final organized board with frames, cards, color coding

### Act 3: The Tech (30s)
"Built with Codex for code generation, Playwright for web automation,
Miro MCP for visual output. The agent reasons, plans, uses tools, and completes
real tasks — the four criteria you asked for."
