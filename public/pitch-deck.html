Not useful.

  • Cross-agency data joins feel like root canal surgery. Permits + 311 + code violations = three portals, three schemas, one nightmare.

Agentic Integration

One stack. Any Socrata portal. Everywhere.

TXLookup isn't a TX-only tool — it's the installable intelligence layer that any Socrata-powered city, county, or state can deploy. The dataset scout auto-discovers new datasets every 6 hours.

01
Bring your own portal
Point TXLookup at any Socrata endpoint. Full agent pipeline, insight cards, and the /q search layer — without rebuilding anything.
02
Install as a Claude skill / MCP server
Ship as a Claude Code skill or FastMCP server. Any Claude user or agent can invoke it — no custom UI required. Anthropic Agents Track ✓
03
Dataset scout · 6h auto-discovery
GitHub Action scans portals every 6 hours, ingests new datasets into SQLite cache, and surfaces them in the agent catalog automatically.
04
Cross-city intelligence
Austin + Dallas + TX State today. San Antonio, Houston, El Paso in the scout queue. One query can span all connected portals simultaneously.
