# TXLookup Brand Guide

Single source of truth for all design, copy, and asset decisions.

## Files

| File | Use |
|------|-----|
| `BRAND.md` | Full brand spec — colors, typography, voice, logo rules, component patterns |
| `txlookup-logo-dark.svg` | Full wordmark on navy (`#0D2340`) — use in dark headers and footers |
| `txlookup-logo-light.svg` | Full wordmark on cream (`#FAF7F2`) — use in light headers |
| `txlookup-mark.svg` | Star mark only (no wordmark) — favicon, app icon, avatar |

## Quick reference

| Token | Hex | Use |
|-------|-----|-----|
| Navy | `#0D2340` | Primary background, hero |
| Rust | `#C4420A` | CTAs, action buttons |
| Gold | `#D48B10` | **TX** wordmark, star, accents |
| Sky | `#3A7FBE` | Links, data nodes, info states |
| Cream | `#FAF7F2` | Page background, light surfaces |

**Fonts:** DM Serif Display (headlines) · Syne (UI/body) · IBM Plex Mono (code/labels)

**Google Fonts import:**
```
https://fonts.googleapis.com/css2?family=DM+Serif+Display:ital@0;1&family=IBM+Plex+Mono:wght@400;600&family=Syne:wght@400;700;800&display=swap
```

See `BRAND.md` for the full spec before touching any component.
